จิ้มส่องตัวอย่างกลุ่มป้าย

ขั้นตอนการสั่งซื้อ 🛒

1. เลือกกลุ่มที่สนใจ แคปส่งให้ร้านได้เลย

2. แจ้งรวมยอด (หากมีหลายกลุ่มรบกวนแคปทั้งหมด)

3. หลังรวมยอด กดไปที่ลิ้งค์ชำระเงิน :

👉 ลิ้งค์ชำระเงินคลิกที่นี่

4. ดูยอดที่ร้านแจ้ง และโอนตามจำนวน

5. ส่งสลิป และ Qr / Id / Link ไลน์ที่จะใช้เข้ากลุ่ม

6. รอมิ้นดึงเข้ากลุ่มภายใน 3-10 นาที ⏳

ขั้นตอนการรันบอท 🤖

1. สังเกตุจุดประกาศของแต่ละกลุ่ม : สิ่งสำคัญจะมี กฎ/กด และ เมล

2. รันบอท : พิมพ์หน้าห้องทีละคำ ห้ามพิมพ์รวบ!
กดเมล❌ | กด✅ เมล✅

3. อ่านกฎให้ละเอียด : หากอ่านไม่จบ อาจพลาดส่วนสำคัญจะไม่โดนดึงเมลนะคะ

4. ลงเมลในโน๊ต : หาโพสต์ที่แอดมินจัดไว้ให้ลงเมล (ต้องเป็น Gmail.com เท่านั้น)

5. คีย์เวิร์ดพิเศษ : บางกลุ่มมีคำสั่งแท็กเรียกแอดมิน ให้อ่านให้ละเอียดค่ะ

⚠️ ทำผิดขั้นตอน ไม่โดนดึงเมล ให้ย้อนกลับไปอ่านใหม่ทั้งหมดอีกรอบนะคะ

กฎการผ่อนกลุ่ม 🧸

🌸 เปิดบิลผ่อน: ไม่มีขั้นต่ำ ส่งยอดได้เรื่อยๆ ไม่จำกัดเวลา

⚠️ ข้อที่สำคัญที่สุด: ยกเลิกผ่อนไม่คืนเงินทุกกรณี ตัดสินใจให้ดีก่อนเปิดผ่อน

• มิ้นไม่ตามยอดนะคะ หากขาดส่งนาน 3 เดือน (ไม่แจ้ง/ไม่ตอบทัก) = ยึดยอดทันที

• บางกลุ่มอาจมีการอัพราคาขึ้น ติดตามและโปรดรับผิดชอบตัวเองด้วยนะคะ

• ผ่อนครบ = ดึงเข้ากลุ่มเท่านั้น

💡 กรณีผ่อนหลายกลุ่ม หากยอดพอเข้ากลุ่มใดกลุ่มนึงได้ก่อน สามารถแจ้งเข้าก่อนได้เลยค่ะ
