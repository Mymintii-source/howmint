จิ้มส่องตัวอย่างกลุ่มป้าย

ขั้นตอนการสั่งซื้อ 🛒

1. เลือกกลุ่มที่สนใจ แคปส่งให้ร้านได้เลย ✨

2. แจ้งรวมยอด (หากมีหลายกลุ่มรบกวนแคปทั้งหมด) 🛍️

3. หลังรวมยอด กดไปที่ลิ้งค์ชำระเงิน :

👉 ลิ้งค์ชำระเงินคลิกที่นี่ 👈

4. ดูยอดที่ร้านแจ้ง และโอนตามจำนวน ส่งสลิป และข้อมูลติดต่อ

5. รอมิ้นดึงเข้ากลุ่มภายใน 3-10 นาที ⏳

ขั้นตอนการรันบอท 🤖

1. สังเกตุจุดประกาศ : สิ่งสำคัญจะมี กฎ/กด และ เมล 📢

2. รันบอท : พิมพ์หน้าห้องทีละคำ

ห้ามพิมพ์รวบ! กดเมล❌ | กด✅ เมล✅ ⌨️

3. อ่านกฎให้ละเอียด : หากอ่านไม่จบ อาจพลาดส่วนสำคัญจะไม่โดนดึงเมลนะคะ 📖

4. ลงเมลในโน๊ต : หาโพสต์ที่แอดมินจัดไว้ ( Gmail.com เท่านั้น ) 📧

⚠️ ทำผิดขั้นตอน ให้ย้อนอ่านใหม่ทั้งหมดอีกรอบนะคะ
